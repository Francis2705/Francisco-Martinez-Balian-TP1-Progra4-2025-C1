Juego propio: Dados Locos

En este juego tendrás dos dados para tirar, y el objetivo será obtener dos seis. Tendrás un boton para lanzar los dados y un cronómetro para que veas cuanto tiempo llevas jugando.

Reglas:
  • Para ganar deberás sacar dos seis en ambas caras.
  • Si sacas una suma menor a 4, pierdes.
  • Puedes tirar los dados tantas veces como desees hasta que se termine el juego.

El diseño es simple y está pensado para jugar de forma rápida. A medida que tires los dados, se te contarán las tiradas para luego guardarlas en el sistema de puntuaciones.

Recomendación: si ya completaste el juego, ¡intenta superar tu marca personal y las de los demás!

En la página resultados encontrarás las puntuaciones de todos los usuarios.
